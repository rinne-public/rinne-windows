# Rinne (Windows) Support

This repository is the public issue tracker for Rinne on Windows.

Website: https://rinne-btn.pages.dev/

## Use GitHub Issues
Please use **Issues** for:
- Bug reports
- Feature requests
- Compatibility reports, documentation fixes, and product feedback

## Crash reports (Windows)
If the app crashes, Rinne writes a crash report under:

- `%LOCALAPPDATA%\rinne\crash\`

Open the newest `crash-gui-*.txt` and include its contents in your issue.
Please redact any sensitive paths or names if needed.
